# Personal Portfolio Website

A clean, modern portfolio website with TypeScript.

## Features
- Fixed left sidebar with profile information (non-scrollable)
- Scrollable right content area
- Static header navigation
- Smooth scrolling between sections
- Responsive design

## Setup

1. Install dependencies:
```bash
npm install
```

2. Build the project:
```bash
npm run build
```

3. Add your profile picture as `profile.jpg` in the root directory

4. Customize the content in `index.html` with your information

## Customization

- Edit `index.html` to update your personal information
- Replace `profile.jpg` with your photo
- Modify `styles.css` to change colors and styling
- Update sections in `index.html` with your education, experience, and publications

## GitHub Pages Deployment

1. Build the project: `npm run build`
2. Push to GitHub
3. Go to repository Settings > Pages
4. Set source to "Deploy from a branch"
5. Select "main" branch and "/ (root)" folder
6. Save and wait for deployment

Your site will be available at: `https://yourusername.github.io/repository-name`
